# GFX

Custom ForneyLab.jl factor node for generalised filtering with exogenous input.

### Usage

The node can be added to ForneyLab by running 

```julia
] dev git@github.com:wmkouw/GFX.git
```

in the REPL and then import

```julia 
using ForneyLab
using GFX
```
